A SwiftUI menu-bar app for macOS that competes in the Magnet/Rectangle slot, with optional AI-driven features on top. Global hotkeys snap the front window to preset positions; a Claude-powered Smart Sessions feature auto-arranges multi-window layouts tailored to coding, design, writing, or research contexts. Privacy-first - only app names and screen dimensions cross the wire, never window content.

Sandbox configuration and feature-gated views are wired end-to-end. An actor-isolated API client pairs with a rule-based fallback so the UI feels instant while the LLM is in flight. Third stablemate in the CrazyAIApps line.
