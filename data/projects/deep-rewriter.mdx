Deep-rewriter is an early-stage exploration to productize an existing personal Claude skill - one that dispatches a panel of specialist and meta-assessment passes over a document, then rewrites it incorporating the feedback - as a standalone paid service. The reference point is reviewer3.com, which monetizes adversarial peer-review of research papers; the same multi-perspective critique loop generalizes naturally to PRDs, design docs, RFCs, and long-form articles.

Status: idea phase. The engine already works as a daily-use skill, so the open questions are productization rather than feasibility: which vertical to target first (researchers, PMs, technical writers), self-serve web vs. API delivery, and whether the model-cost economics support a viable per-document price.
