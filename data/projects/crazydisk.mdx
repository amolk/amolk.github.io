## About

CrazyDisk is a native macOS app that scans volumes and folders, visualizes usage via an interactive sunburst, and suggests what to delete. Built in Swift / SwiftUI (macOS 14+), packaged as both a GUI app and a `crazydisk` CLI sharing the same scan core. Productized for App Store sale at $4.99/yr with a 30-day trial and a landing page. Active development from Mar–Apr 2026 with rapid weekly polish iterations.

## Why notable

- Real shipping product, not a prototype: pricing, trial, landing page, App Store path.
- Clean separation of scan core / suggestion engine / CLI / GUI — same Swift packages drive both surfaces.
- Tackles hard macOS edges: APFS snapshot deletion identity, hidden-space accounting, security-scoped bookmarks, transactional deletes.

## Architecture

- `Package.swift` — Swift Package (tools v5.9, macOS 14+), depends on swift-argument-parser; defines `crazydisk` CLI executable and shared targets.
- `shared/` — reusable Swift modules: `CrazyScanCore` (scan engine), `SuggestionCore` (cleanup heuristics with bundled `rules.json`), `SessionInterchange` (CLI ↔ app handoff).
- `app/` — SwiftUI macOS app (`CrazyDisk.xcodeproj`, `project.yml` for XcodeGen) with `LandingView`, sunburst visualization, starred-folders store.
- `cli/Sources/crazydisk` — command-line front-end built on the same cores.
- `website/` — liquid-glass landing page with interactive carousel.
- Build via `make` (root `Makefile`) for CLI, Xcode/XcodeGen for the app.
