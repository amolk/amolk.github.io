A personal knowledge graph of every project I work on, past and present - one Markdown page per project with typed frontmatter, wikilinks between related work, people, and concepts, and dated activity logs.

What makes it fun is that it maintains itself: an LLM agent periodically reads my repositories, reconciles what changed across the graph, and rewrites the affected pages. This portfolio is a downstream view - the public site renders the subset of the graph I choose to make visible.
