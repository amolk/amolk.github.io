amolkelkar.me is this site - my personal portfolio. It's a Next.js 15 + React 19 static export where every project is rendered from a typed MDX file via Contentlayer, deployed to GitHub Pages through a GitHub Actions workflow. The domain has been my home on the web since 2018; in 2026 I rebuilt it from scratch into this portfolio.

The pages here aren't hand-authored one by one - project entries are generated from a separate [knowledge graph](/projects/llm-wiki-vault/) I keep of everything I build, so the site is really a public, rendered view of that graph.
