Bertrand-DR (Bertrand Discriminative Re-ranker) was a 2020 research effort: a discriminator-based re-ranking approach for improving neural text-to-SQL parsers on the [Spider](https://yale-lily.github.io/spider) benchmark. Rather than generating SQL directly, the method trains a separate discriminator to score candidate parses from a base model and re-rank them, lifting accuracy on hard queries.

Honest caveat: this code repository never landed - only a README placeholder. The actual research artifact (method, results, write-up) lives in the companion bertrand-paper repo, with supporting tooling in text2sql-utils. The portfolio value here is in the paper, not the code.
