## About

Intended home for the Bertrand-DR (Bertrand Discriminative Re-ranker) model, a discriminator-based re-ranking approach for improving neural text-to-SQL parsers on the Spider benchmark. The code repo never landed — it contains only a README with a "coming soon" note. The actual research artifact (method, results, write-up) lives in bertrand-paper, and the supporting tooling in text2sql-utils.

## Why notable

- Companion code to a real research write-up on the [Spider](https://yale-lily.github.io/spider) leaderboard line of work.
- Honest caveat: this repo itself is empty — portfolio value is in the paper, not the code.
