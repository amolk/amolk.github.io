## About

A Rails 3.2.8 scaffold generated September 2012 via `rails_apps_composer`. Despite the name, no YouTube ingestion or stats logic was ever built — `app/models/` is empty and only a `home_controller` with an index view exists. Single commit in history.

## Why notable

- Honestly: very thin — a placeholder for an idea that never materialized. Consider deselecting from portfolio; the name promises more than the repo delivers.

## Architecture

- Rails 3.2.8 on Ruby (no explicit pin), MySQL.
- Generator stack: Bootstrap/Sass + Haml, RSpec + Capybara + FactoryGirl, Simple Form, Gmail email, WEBrick dev server.
- No authentication, no models, two controllers (`application_controller`, `home_controller`).
