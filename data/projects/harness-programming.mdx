## About

The book draws two parallels:

| Broad discipline       | Narrower sub-discipline                       |
| ---------------------- | --------------------------------------------- |
| Software Engineering   | Computer Programming                          |
| Harness Engineering    | **Harness Programming**                       |

The second parallel: just as conventional programming diversified into schools (functional, logic, OO) each demanding its own runtime, natural-language programming will diversify into schools, each requiring corresponding harness capabilities. Playbooks is treated as the clearest concrete example of a structured-DSL school of natural-language programming, and as the strongest argument for why filling in formal constructs is a bet against the model getting better.

Target ~25K words, ~2–3K per chapter. Chapters self-contained, readable out of order.
