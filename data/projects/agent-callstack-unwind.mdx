## About

unwind reads `~/.claude/projects/*.jsonl` (Claude Code's session logs) plus `<project>/.claude/callstack/log/` and renders, in a local web UI, every session that has run in a project — the conversation for each, the call hierarchy between sessions when callstack's `/call` skill is used, and any spawned subagents. Three-pane layout: project sessions list, call-tree canvas, conversation detail. `unwind serve` from a project folder opens a browser tab; same data is also available from `unwind project|session|messages|task` CLI subcommands.

**Pivot history.** Originally scoped as a Rust ratatui TUI with continuous retrospective in-process compaction. On 2026-04-27 the Rust workspace was deleted and a Python observability tool (`unwind-lite`) was renamed in to become unwind. The retrospective-compaction work is deferred to a v2 release (agent-callstack-compaction-paper follows that release).

## Architecture

Stack: Python 3.11+, FastAPI + uvicorn (server), watchdog (file watcher), custom shadcn-style renderer for the canvas. Build backend `poetry-core`; tag-triggered PyPI publish via OIDC trusted publisher.

```
unwind/
  src/unwind/           Python package
  web/                  vite build output served by FastAPI
  dev/                  PLAN.md, PRD.md, smoke.sh
  tests/                pytest
  pyproject.toml        poetry-core, PEP 621 [project]
```

**PyPI naming quirk:** distribution is `unwind-labs` (the `unwind` name is owned by an unrelated package), but the installed console script is `unwind` and the import is `import unwind`. Install: `pip install unwind-labs` → `unwind serve`.
