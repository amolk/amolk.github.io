InfoPath Forms Server brought InfoPath forms to the browser - rendering the same XML-backed forms server-side as HTML/JavaScript so they could be filled by anyone without the desktop client installed. This was a substantial engineering effort to faithfully reproduce a rich desktop forms experience across browsers, and it became part of the SharePoint/Office Server stack.
