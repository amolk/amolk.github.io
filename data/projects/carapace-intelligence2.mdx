A deterministic Python implementation of a Decision Intelligence framework laid out in a separate paper. The LLM is treated as a learned environment model - a stochastic transition operator and reward elicitor - and explicit search, Bayesian expectimax backward induction, and structured robustness analysis are layered on top. All global reasoning (value, recommendation, sensitivity) is deterministic and reproducible from cached LLM outputs, with mandatory rationale sidecars on every cache entry.

Built as a v2 rethink of an earlier exploration. The original April workspace was a "query-first, markdown-native" sketch that sat dormant for a month; the current codebase was bootstrapped in a single overnight run that took it from empty repo to eight end-to-end phases - frozen-dataclass types and content-addressed cache, deterministic solver with subopt/best-line/opponent modes, four-arm sensitivity sweep, self-validating elicitation agents, full-BFS tree builder, tail-risk audit gate, prescriber, orchestrator and CLI - with 107 tests passing and ruff clean.
