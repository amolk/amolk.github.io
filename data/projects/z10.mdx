## About

Zero-10 is a design tool where AI agents edit designs through standard DOM APIs. The `.z10.html` file format annotates plain HTML/CSS with `data-z10-*` attributes and embedded JSON metadata, so agents can read and write UI natively — no custom SDK, no brittle schema. A web canvas (Next.js + Postgres) streams edits live; a CLI and MCP server give agents a minimal DOM-style surface (`createElement`, `setAttribute`, `style.*`). The z10 Claude Code skill wraps this so an agent can design full dashboards from chat.

## Architecture

```
src/
  core/       Z10Document model, command executor
  cli/        `z10` binary — serve, new, info, dom, exec, component, tokens
  mcp/        MCP server exposing read/write tools to agents
  format/     .z10.html parser + serializer
  dom/        DOM mutation + transaction layer
  export/     export targets
web/          Next.js canvas app + REST API + Postgres (via Drizzle)
skills/z10/   Claude Code skill — wraps CLI for agent use
tests/        Vitest suite
```

Run: Postgres → `cd web && npm run dev` → `z10 login --token <tok> --server http://localhost:3000`
