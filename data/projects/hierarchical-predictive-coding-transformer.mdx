## About

PyTorch research codebase that fuses **hierarchical predictive coding** with transformer architectures. The core idea: train transformer layers as a stack of predictors, each one predicting the next layer's representation and propagating only the residual (prediction error) — a structural analogue to biological predictive coding. Five model variants explore complementary mechanisms — confidence-gated early exit (CAE), difference target propagation (DTP), Hebbian updates, and locality constraints — against a standard ViT/BERT baseline. Active solo research; CIFAR-10 / ImageNet / GLUE training and analysis pipelines all in place, GPU-grade run still pending for full accuracy validation.

## Why notable

- Original ML research with five distinct architectural variants, not a fork of an existing repo.
- End-to-end engineering rigor: configs, trainers, tests, ablation scripts, W&B integration, smoke tests — not just a notebook.
- Documented learnings file (`fix_plan.md`) shows scientific honesty about bugs found and validated (CAE confidence saturation, HERT gradient isolation, residual magnitude decay across stages).

## Architecture

- `src/models/` — `vit.py`, `bert.py` baselines plus `hert.py`, `hert_cae.py`, `cae.py`, `dtp_hert.py`, `hebbian_hert.py`, `locality_hert.py`, `early_exit.py`, plus a `factory.py` dispatcher.
- `src/training/` — one trainer per variant (`hert_trainer`, `cae_trainer`, `hert_cae_trainer`, `dtp_trainer`, `hebbian_trainer`, `locality_trainer`, `bert_trainer`) sharing a base `trainer.py`.
- `configs/` — YAML configs for CIFAR-10, ImageNet, GLUE, and HERT/CAE combinations.
- `scripts/` — 12+ analysis scripts (attention, CKA, calibration, t-SNE, scaling laws, early-exit depth, residual amplitudes, ablations) plus per-variant `train_*.py` entry points and `smoke_test.py`.
- `tests/` — pytest coverage for HERT, CAE, HERT+CAE, early-exit, mixup, warmup scheduler, gradient clipping, ViT.
- `wandb/` + `checkpoints/` — Weights & Biases experiment tracking; staged HERT+CAE checkpoints up to `hert_cae_stage_final.pt`.
